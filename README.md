### Installation

- Clone this repository -
```
$ git clone --depth=1 https://github.com/AlessandroPerazzetta/ricing-hyprland.git
```

- Change to cloned directory and make setup.sh executable -
```
$ cd ricing-hyprland
$ chmod +x setup.sh
```

- Run `setup.sh` -
```
$ ./setup.sh
